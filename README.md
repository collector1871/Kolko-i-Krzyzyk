# Kolko-i-Krzyzyk
Kółko i krzyżyk w wersji tekstowej - wielkość 3x3. Język - czyste C.  
Z komputerem nie da się wygrać: możliwy jest tylko remis lub przegrana.

### kompilacja i uruchomienie - 2 komendy (Windows-Cygwin lub Linux):

	make
	./kik.exe

Samo uruchomienie:

	./kik.exe
